How To: Import Transactions

The first step is to download your account activity from your online banking website. Do this by using your web browser (such as Chrome, Firefox, Safari or Internet Explorer) to login to your bank's online banking website.

Decide which accounts you'd like to include, then follow the following steps for each account:

  1. Select the account.
  2. Select a time period.
  3. Select the download file format.
    Supported file types are "Microsoft Money", "Intuit Quicken", "Intuit Quickbooks", or "Simply Accounting".
    "CSV", "Excel" and "Spreadsheet" file types are not supported.
  4. Press the Download button.

Repeat the above steps for each account and time period of interest. Once you've downloaded the account activity for each account, import the corresponding files into {{applicationName}} as follows:

  1. Using {{applicationName}} select "Import" from the main menu.
  2. Press the "Select Files..." button.
  3. Select all of the account activity files that you downloaded from the bank. You can select multiple files at once.
  4. Press the "Open" button and then the "Import Files" button.

Tip: Importing Multiple Times

It's okay to import the same account activity multiple times. {{applicationName}} will automatically detect that transactions have been previously imported and ignore duplicates. This makes it easy to import account activity from the current month or statement more than once as the days go by to get the most up to date information without having to worry about duplicates.

Did You Know: Encryption

Banking information is private, and we want to keep it that way! By encrypting your data using strong encryption you're taking steps to ensure that it's safe from prying eyes.

Data imported into {{applicationName}} is encrypted using 256-bit AES encryption with Galois Counter Mode (GCM). The secret key is derrived using PKCS #5's PBKDF2 (Password-Based Key Derivation Function #2). The password is salted to prevent dictionary attacks against the key. A random 16-byte initialization vector is applied to ensure that each encrypted message is unique.

While encryption does not 100% guarantee data safety, strong encryption is the best known way to protect your data.

How To: Categorize Transactions

Categorizing transactions enables you to view your expenses and income for each category. To categorize transactions from your imported account activity, follow these steps:

  1. Using {{applicationName}} select "Transactions" from the main menu.
  2. Select any month having transactions that you'd like to categorize.
  3. For each transaction, hover over the "Category" column and click on the "Choose category..." link.
  4. Select the appropriate category, and
    • press the "This Transaction Only" button to apply the cateogory for the transaction, or
    • edit the description as needed to match multiple transactions, and press the "Matching Transactions" button to apply the category to every transaction that matches the desription.

Tip: Apply Category to Multiple Transactions

A category can be applied to every transaction that matches a description. Edit the description text by removing unimportant information such as leading or trailing numbers. For example, to categorize all transactions matching "HOME DEPOT" with a description of "HOME DEPOT #12355", remove "#12355" to leave a description of "HOME DEPOT".

Tip: Sorting Transactions

The transactions for any month can be sorted by date, category, income or expense.

Sorting transactions by category can help to find uncategorized transactions, and can help to provide an understanding of which transactions contribute to a specific category.

Sorting transactions by expense can help in identifying the largest expenses.

How To: View Reports

Reports will help you to understand your cash flow. Using {{applicationName}} select "Reports" from the main menu.

Report: Monthly Income v.s. Expenses

The Monthly Income v.s. Expenses report provides a high level view of month-to-month spending and how it compares to income.

Tip: Month Totals

Hover over any month in the report to see the total income or expenses for a specific month.

Tip: Month Breakdown by Category

To see a breakdown of expenses by category, click on any month in the report.

Report: Monthly Expenses by Category

The Monthly Expenses by Category report provides a comparison of month-to-month expenses by category. Up to 4 categories can be selected for comparison.

Tip: Month Totals

Hover over any month in the report to see the total expenses for each category for a specific month.

Tip: Month Breakdown by Category

To see a breakdown of expenses by category, click on any month in the report.
