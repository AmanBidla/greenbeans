What To Do?

This application works by importing transaction data that you provide. To get the files that you need, you'll have to log in to your bank website with your web browser and follow these steps:

  1. First, using your web browser log in to your bank website.
  2. For each account that you want to track transactions, follow the following steps.
  3. Select an account.
  4. Select a time period.
  5. Select the download file format. (Should be "Microsoft Money", "Intuit Quicken", "Intuit Quickbooks", or "Simply Accounting")
  6. Press the Download button.

When you're done downloading transactions for every account and time period of interest, use the "Select Files" button above to import the downloaded files.
